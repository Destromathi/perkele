README
1. Valvonta: Tarkastaa tiedostojen määrän ja tallentaa sen loki.txt
2. varmuuskopio: Varmuuskopioi kansion ja sen sisällön ja tallentaa sen loki.txt
3. luokansio: Luo kansion ja tallentaa sen loki.txt
4. etahallintascript: Hakee etä koneen nimen ja ajan ja tallentaa sen loki.txt
6. levytilan-tarkistus: Tarkastaa vapaana olevan levytilan (GB) ja tallentaa loki.txt
7. paikallinenajo: Suorittaa komentoja ja skriptejä etäkoneilla. Ei tallenna loki.txt
8. kayttajaraportti: Hakee paikallisten käyttäjien tiedot, tallentaa ne CSV-raportiksi Raportit-kansioon ja kirjaa onnistumisen tai virheen lokiin.
9. funktiot: sisältää funktiot mitä kaikki ps1 voivat käyttää mukaan lukien loki.txt tallentaminen
